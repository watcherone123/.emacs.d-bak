# deadfd
在windows 下搜索文件很不方便，之前一直用Everything,效果不理想。
在网上发现deadgrep,觉得可以将deadgrep改造一下，用fd搜索文件。
deadfd主要在windows10使用
# 用法
M-x deadfd
<img src="./images/fd-screenshot.png">
# 方法
| 名称      | 描述            |
| :-------- | :----                    |
| deadfd    | 搜索输入字符在默认文件夹 |
| fd        | 自定义搜索               |
# 快捷：
| 快捷键    | 描述                              |
| :-------- | :----                             |
| Enter     | 用windows默认程序打开             |
| o         | 用windows默认程序打开             |
| f         | 用windows默认程序打开文件所在目录 |
